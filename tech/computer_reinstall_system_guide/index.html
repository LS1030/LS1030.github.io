电脑安装系统指南

 2019.8.12  2022.5.19  Tech  3344  7 分钟

#电脑安装系统指南

#简介

安装系统是很常见的操作，刚组的新电脑需要安装系统，旧电脑出现未知的BUG或者莫名其妙地蓝屏的时候，最简单明了的方法也就是直接重装系统，重装之外所需要的仅仅是就是备份资料和重新安装软件，简单直接。以下就是安装系统的详细操作。

#前期准备

重装前所需的准备有以下几点：

  • 系统镜像
  • 电脑相关驱动
  • 8G以上的U盘(非必须，但一般都要)
  • 激活工具(Windows需要)
  • 常用软件安装包(非必须，但最好准备一下以免重新下载)

系统原版镜像 从网上下载就行，Windows可以从MSDN_itellyou、MSDN_next下载，Linxu可以从各发行版的官网下载，比如Ubuntu、CentOS、Deepin。

电脑驱动最重要就是网卡驱动，其他驱动可以在联网后用其他软件进行安装，比如鲁大师、驱动精灵。

下载驱动的地方最靠谱的当然是官网，比如说Intel、瑞昱。第三方的网站也有比较靠谱的，比如驱动之家。

在官网或者第三方网站搜索相关驱动首先要了解驱动型号，驱动检测软件除上面的鲁大师、驱动精灵之外，AIDA64也不错。

#安装步骤

#备份资料

一般来说，重装系统仅需要备份系统盘(C盘)的资料，将C盘的文档、图片、视频备份到其他盘即可，安装在C盘的软件在重装系统后重新安装，仅需备份配置文件，无需完整备份（因为如果不重新安装的话，采取先备份到其他盘，在重装系统后复制到系统盘的方式，可能会遇到注册表、环境变量等问题，重新安装比较快）。

但碰到其他一些特殊情况，比如说需要将MBR磁盘转换为GPT磁盘的情况，可能要把全盘的个人文档全部备份，此时就需要大容量的U盘或者移动硬盘了。

#安装系统

安装系统有三种方法：

  • 硬盘安装
  • U 盘安装
  • 光盘安装

#硬盘安装

硬盘安装适用于可以正常开机的系统，也就是仅适用于系统重装，仅适用于安装Windows。

首先先用光盘刻录软件UltraISO、或者解压缩软件bandizip、7zip、360zip，将下载后的镜像完整解压到C盘以外的文件夹中。进入文件夹，直接点击Setup.exe，然后按步骤操作即可。

可能遇到分区的问题，在资料备份完毕后，直接把C盘的分区全部删除，让Windows自动新建分区就行，注意，此操作会删除C盘的所有数据，当然，为了保险起见，一般C盘的大小为100G以上比较好。

安装完毕后会自动重启，之后激活并且安装相应软件，重装完成。

#U盘安装

U盘安装适用于任何系统，包括新系统的安装和老系统的重装，安装Windows和Linux都可以。

#制作装机U盘

制作装机U盘可以使用系统镜像烧录或者是PE的方式。

系统镜像可以用Rufus或者UltraISO将原版镜像刻录到U盘中，此过程需要格式化U盘，注意备份U盘文件。 或者可以自行格式化U盘为FAT32格式(考虑到兼容性一般使用FAT32)，再把解压后的镜像的所有内容拷到U盘中。

PE可以实现更多功能，需要较复杂的操作(分区、转换磁盘)的时候会选择用PE的方式。 PE可以选择微PE、老毛桃、大白菜。由于考虑到广告问题，一般采用比较纯净的微PE，但是微PE对较老的电脑支持不太好，老电脑重装可以考虑使用后两者。 下载完PE后，按各PE的操作直接制作装机U盘即可。

#BIOS设置U盘启动

按正常情况，BIOS默认设置从已经装好系统的硬盘进行启动。此时，应进入BIOS设置从U盘进行启动。

#进入BIOS

进入BIOS的方法是在开机后的5秒内（系统未启动之前）按下相关热键进入，此热键根据主板的生产厂家有所不同，有F2, F7, F10, F12, Del, Esc, Enter等等。

各品牌电脑(主板)的BIOS热键（第一个常用）:

  • 戴尔Dell: F2, F12, Del
  • 惠普HP: F10, Esc
  • 联想Lenovo: Enter, F1, Del, F2
  • 华硕Asus: F2
  • 宏碁Acer: F2
  • 华擎、微星、技嘉: Del

综上，常用的热键一般是Del, Esc, 可以同时反复按进行尝试。

以上热键是进入BIOS进行详细设置所用，某些厂商专门为启动项更改提供专门的热键，比如微星的F11，华硕的F8，技嘉的F12等，此时直接用上(↑)下(↓)键和Entel选择U盘即可，可跳过之后的BIOS设置这一步。

#修改系统启动项

进入BIOS后，用上下左右键进入Boot菜单，首先选择启动方式Boot Mode，一般来讲，装win7或者winXP等老系统并且是MBR磁盘选择legacy，装win8或者win10并且是GPT磁盘用UEFI模式，当然也有在UEFI+GPT模式下装win7的骚操作，在此先略过不提，此两种模式一般都能读到装机U盘，若读取不到就只能用legacy了。

选择完启动方式Boot Mode后，对启动项进行排序，将U盘启动放在第一位。 在启动优先级设备选择Boot Device Priority的第一启动设备选择装机U盘。

设置完毕之后，按F10保存操作并且自动重启。

#进入安装界面安装系统
#系统镜像

若是采用系统镜像烧录的方式制作的装机U盘，重启完毕后，进入到系统安装界面，选择自定义方式，然后进行分区。

若是原有的系统进行重装，并且C盘是独立的磁盘（比如是固态+机械的电脑）,直接把C盘的分区删除，让系统自动分配C盘的分区就行。**注意：此操作会删除C盘的所有数据。**若C盘和数据盘是同一个磁盘（纯机械或者纯固态的电脑），也可以用上述方式，只是需要小心操作避免删除其他盘的数据。

当然也可以手动进行分区，不过比较麻烦，一般来讲，C盘分区分100G左右就可以了，情况允许的话越大越好，分区后最好格式化后再装系统。

接下来就一路确定等待重启就行，第一次重启之后一般是已经使用完装机U盘了，可以进BIOS调整开机启动项为硬盘开机或者直接拔下U盘。

#PE方式

采用PE方式制作的装机U盘，重启完毕后，会进入到对应的PE界面。在PE界面可以采取很多操作，下面以微PE为例讲解如何在PE下装系统。

在微PE的界面下，以下常用工具：

  • Dism++：用于系统优化
  • 分区助手、DiskGenius；用于磁盘分区、磁盘转换、数据备份
  • UEFI引导修复、BOOTICE： 修复、调整引导启动项
  • HDTune 硬盘检测：硬盘检测及数据擦除
  • Windows安装器：安装Windows

新电脑装系统一般需要先分区，可以选择用分区助手或者DiskGenius，分区助手比较可靠，DiskGenius功能更强大但偶尔会出问题，为了不丢失数据，一般选择用分区助手。

分区的话，主要看个人使用习惯，一般情况下，系统盘（C盘）要保证在100G以上，把系统、驱动、软件装上100G已经差不多了。采取128G/256G固态+1T机械的电脑，推荐固态不分区当系统盘；若是纯固态的电脑（500G以上），推荐分两个区，一个系统盘一个数据盘；若是纯机械的电脑，系统盘分150G，数据盘按使用习惯分2~3个区就行了。

在常规的可见盘（C盘D盘等）以外，还需要对不可见盘进行分区操作。不可见盘有:

  • EFI(ESP)系统分区：EFI system partition，UEFI启动必须有此分区，一般分100MB就行。
  • MSR分区：Microsoft Reserved Partition，保留分区，保存分区表数据，必须有此分区，一般大小为128MB。
  • WRE分区：恢复分区，用于恢复或修复系统，可以没有此分区。

分完区后，就可以装系统了，可以选择用Windows安装器进行安装，选择镜像文件，选择引导分区(ESP分区)，选择安装分区(C盘)，选择系统版本，然后开始安装。

此时安装器就开始将系统直接部署到选定的安装分区中，可以理解为直接把文件复制进去。安装完毕后，重启进入系统即可(可以先进BIOS改启动项)。

#光盘安装

光盘安装比较现在比较少见，整体过程和U盘安装大同小异，只不过由U盘变成光盘而已。

首先先准备一张装机光盘，内部刻录有原版镜像或者PE。

然后将光盘放入光驱，进入BIOS，选择从光盘启动，重启。

重启后进入系统安装界面或者是PE界面，直接参考U盘安装的相关内容安装系统就行。

#安装完毕

如无意外，重启后进入了系统，系统安装完毕。

对于Windows系统而言，进入系统之后还不够，需要激活。Windows激活工具有很多，一般采用KMS方式激活，推荐使用神龙版KMS激活工具。

激活后，就可以打上相关驱动，安装常用软件了，推荐把所有软件直接安装到C盘。打完驱动，安装完软件，就可以正常使用了。
