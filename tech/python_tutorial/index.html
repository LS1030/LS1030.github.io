Python入门不完全指北

 2022.7.25  2022.7.25  Tech  2176  5 分钟

#简介

人生苦短，我用python。得益于简洁易懂的语法，数量众多的轮子，python迅速地成为了第一受欢迎的编程语言。以下是简单的python入门指南，主要是包括了安装及环境配置、基本语法、常用库、学习参考资料。主要参考了这可能是一篇最贴心的Anaconda指北了[1]、conda常用命令[2]、Python 官方教程[3]。

#安装及环境配置（windows）

#安装IDE

python编程的IDE一般选择VSCode或者PyCharm。因为PyCharm是商业软件，不能免费使用，因此选择VSCode。可以从VSCode官方下载页面下载对应版本的VSCode，安装完毕后再安装下面的插件：python、Pylance即可。

#安装python（解释器、包管理器、虚拟环境）

安装python解释器时一般都会同时包管理器和虚拟环境。因为python的包非常多，因此需要比较好的包管理器来管理包，同时要创建虚拟环境来做好环境隔离，比较常用的包管理器是pip、conda，它们的区别为[4]：

类别 conda pip
管理 二进制 wheel 或源码
需要编译器 no yes
语言 any Python
虚拟环境 支持 通过 virtualenv或venv等支持
依赖性检查 yes 屏幕提示用户选择
包来源 Anaconda repo和cloud PyPi

常用的虚拟环境有conda、venv、virtualenv、pipenv等，它们的区别可以参考stackoverflow下关于此问题的回答What is the difference between venv, pyvenv, pyenv, virtualenv, virtualenvwrapper, pipenv, etc?[5]。

由于conda集成了包管理器和虚拟环境，使用时比较方便，因此选择使用conda。安装conda时可以选择Anaconda或者Miniconda，Anaconda包含了除conda本身之外的一些科学计算包，显得比较臃肿，而Miniconda仅包含了conda本身和conda'依赖的内容，比较精简，它们之间的区别见Anaconda or Miniconda?。为方便起见选择Miniconda。但需要注意，Anaconda和Miniconda均是商业软件，仅仅是个人免费版才可以自由使用，不可以商用，详见Anaconda 文档。

#安装Miniconda

首先是下载安装包并安装，从Miniconda下载页下载Windows版本的Miniconda3，目前(2022-07-25)最新版本为Miniconda-python3.9。 下载完毕后，安装，注意在Advanced Options里的Add Miniconda3 to my PATH environment variable 不勾选，等安装完成后再手动添加环境变量，需要添加的环境变量为（默认安装路径）：

1
2
3

C:\ProgramData\Miniconda3
C:\ProgramData\Miniconda3\Scripts
C:\ProgramData\Miniconda3\Library\bin

之后，打开PowerTerminal，输入conda --version，输出conda的版本号，安装完成。

注意，早期的conda官方源不稳定甚至需要科学访问，因此早期的许多教程都有修改conda源的操作，但目前conda官方源可以稳定访问，就不需要这一步了。

#创建虚拟环境和安装工具包

conda安装完毕之后，默认是base环境。

conda创建新环境的命令为：

1

conda create -n [env_name] python=[x.x]

conda 安装工具包的命令为(官方源)：

1

conda install -c conda-forge [pkg_name]

conda也可以安装本地包：

1

conda install --use-local [pkg_path]

另外，conda中也可以使用pip来安装工具包，需要先安装pip：conda install -c anaconda pip，然后再使用pip install安装对应的工具包。

#conda常用操作

conda的常用操作有：

  1. 列出当前存在的环境

    1
2

     conda env list
 conda env -e

  2. 切换环境

    1
2

    conda activate [env_name]
conda deactivate [env_name]

  3. 删除环境

    1

    conda remove -n [env_name]

  4. 删除某个包：

    1

    conda uninstall [pkg[=version]]

  5. conda清理(清楚缓存及不用的包)

    1
2
3

    conda clean -p      // 删除没有用的包
conda clean -t      // 删除tar包
conda clean --all   // 删除所有的安装包及cache

  6. conda环境打包及迁移，可参考官方教程[6]：

    1
2

    conda env export environment.yml // 导出环境包列表
conda env create -f environment.yml // 读取文件内容并恢复环境

#配置IDE

安装完Miniconda后，会自带一个python环境base，接下来配置一下VSCode就可以完成了。

首先打开一个py文件，比如说hello_world.py，然后在VSCode的右下角会有一个选择解释器的页面，或者按F1然后搜索并点击Python: Select Interpreter，然后选择对应的环境base。一般情况下会直接搜索到，如果没有的话数据具体路径，如C:\ProgramData\Miniconda3\python.exe或者C:\ProgramData\Miniconda3\env\env_name\python.exe。然后按F5就可以直接调试运行了。

还可以创建launch.json来进行进一步配置：进入右边栏的运行与调试，然后点击创建launch.json，选择python文件，然后就会在当前文件夹下新建.vscode/launch.json文件了。一般来说，把justMyCode设置为false就可以在调试过程中进入工具包内的代码，更多的设置请参考官方教程[7]。
