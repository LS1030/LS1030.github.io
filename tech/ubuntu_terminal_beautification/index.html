#zsh

Shell是Linux/Unix的一个界面程序，负责用户与系统内核的交互。一个Linux系统可以有多种Shell，可以通过命令查看系统内的Shell:

1

cat /etc/shells

常用的 Linux 系统和 OS X 系统的默认 Shell 都是bash，Ubuntu也是。

zsh一个功能非常强大的Shell，兼容bash并且有bash之外更多的功能。

在Ubuntu里一般没有带zsh，可以通过命令安装：

1

sudo apt install zsh

安装完毕之后，需要把bash切换成zsh：

1

chsh -s /bin/zsh

然后关闭当前shell，重新打开新的shell，就可以使用zsh了。

#oh-my-zsh

第一次使用zsh需要进行配置，但是这个配置非常麻烦，我们使用oh-my-zsh进行配置，因此第一次进入时选择0，即仅生成zsh的配置文件:~/.zshrc。

oh-my-zsh是开源的用于管理zsh配置的框架，可以使用不同主题，添加各种插件来增添更多功能，其Github仓库为ohmyzsh/ohmyzsh。

oh-my-zsh使用命令安装：

1

sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"

安装完毕后，oh-my-zsh会把适用于它自己的模板文件添加到~/.zshrc，完整提示如下：

1

或者

1

omz update

#配置

zsh、oh-my-zsh安装完毕后，切换主题和安装一些插件会让zsh更好用。

安装主题或插件后，更新zsh配置文件，即可应用最新的更改：

1

source ~/.zshrc

#powerlevel10k主题

oh-my-zsh有许多主题，其中一个比较好用美观的是powerlevel10k，强调速度、灵活性和开箱即用的体验，Github仓库是romkatv/powerlevel10k。

在oh-my-zsh下安装powerlevel10k可以直接通过命令完成：

1

git clone --depth=1 https://github.com/romkatv/powerlevel10k.git ${ZSH_CUSTOM:-$HOME/.oh-my-zsh/custom}/themes/powerlevel10k

中国大陆用户可以使用 gitee.com 上的官方镜像加速下载:

1

git clone --depth=1 https://gitee.com/romkatv/powerlevel10k.git ${ZSH_CUSTOM:-$HOME/.oh-my-zsh/custom}/themes/powerlevel10k

然后在~/.zshrc中设置ZSH_THEME="powerlevel10k/powerlevel10k"，然后source ~/.zshrc，根据提示进行设置，设置完成就可以使用了。

如果需要重新配置，则使用命令重新配置：

1

p10k configure

另外，因为主题内含有图标，所以需要改终端字体为包含图标的字体，推荐DejaVuSansMono NF、JetBrainsMono NF、Meslo LGM NF。

#插件

在~/.zshrc里的plugins内添加字段可以使用各种插件，有一些插件时zsh默认安装，也有一些是需要另外手动安装的，我的插件配置为：

1

plugins=(git extract z sudo safe-paste zsh-syntax-highlighting zsh-autosuggestions)

#git

git是默认安装并开启的git插件，可以使用各种alias别名来完成git操作，并且在终端上显示git信息，查看alias可以使用命令:

1

cat ~/.oh-my-zsh/plugins/git/git.plugin.zsh

#extract

extract是默认安装的自动解压插件，可以根据压缩文件后缀来自动调用对用的解压软件来解压文件，其使用方法为：

1

x [*.zip/tar.gz/tar.xz]

#z

z是默认安装的目录自动跳转插件，可以自动记忆曾经进入过的目录，并且通过z命令模糊匹配并直接跳转，使用方法与cd类似，但输入目录地址时可以通过tab快速调整。

#sudo

sudo是默认安装的，其作用是连按两次Esc即可在当前命令前自动添加sudo

#safe-paste

safe-paste是默认安装的，顾名思义，安全粘贴，其作用是使粘贴的命令到终端后需要手动运行，防止复制命令时自带换行导致自动运行不正确的命令带来不可预料的后果。

#zsh-syntax-highlighting

zsh-syntax-highlighting是一个命令自动语法高亮插件，需要手动安装，安装命令为：

1

git clone https://github.com/zsh-users/zsh-syntax-highlighting.git ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-syntax-highlighting

#zsh-autosuggestions

zsh-autosuggestions是一个自动补全插件，根据历史记录自动提示并可以使用tab自动补全，需要手动安装，安装命令为：

1

git clone https://github.com/zsh-users/zsh-autosuggestions ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-autosuggestions

  2. 安装oh-my-zsh

  4. 安装并配置对应插件
